# Kuzzle Application

_An application running with [Kuzzle](https://github.com/kuzzleio/kuzzle)_

## Installation and run

Requirement:
 - Node.js = 18
 - NPM = 10.1.0
 - Docker
 - Docker-Compose

First, install [Kourou](https://github.com/kuzzleio/kourou), the Kuzzle CLI: `npm install -g kourou`

# Usage

```bash
docker compose up -d
```
